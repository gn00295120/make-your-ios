MakeYour Support

Help for building and using your private native tiny apps.

Contact

Email support@claude-world.com. Include the MakeYour version, iOS version, feature, and exact error text. Never send an API key or private content.

Where is my API key?

Your OpenAI key is stored in iOS Keychain with When Unlocked, This Device Only protection. Remove it from AI Key → Remove. An optional Twelve Data key for symbols beyond its AAPL demo uses the same protection and can be removed from the market provider-key screen.

Generation is not working

Confirm your key is valid for the selected Responses API model, Allow requests to OpenAI is enabled, and the iPhone is online. Provider errors appear in the app without printing the key.

Are FX values live market quotes?

No. They are the latest available daily reference rates from Frankfurter. They are not streaming quotes or financial advice.

Where do news and stock data come from?

News uses fixed BBC World, BBC Technology, and NPR News RSS feeds; local search, topics, and bookmarks are not sent to those providers. Market data uses fixed Twelve Data endpoints. AAPL supports demo access; other symbols require your own Twelve Data key. Quotes can be delayed or rate-limited and are not financial advice.

Why is a device action unavailable?

Camera capture, live QR/barcode/text scanning, pedometer data, on-device speech models, and meaningful haptics require supported physical iPhone hardware and may be unavailable in Simulator. Camera, microphone, Speech Recognition, location, and motion also require the corresponding iOS permission. MakeYour starts these actions only after you tap.

What can device actions access?

Scanning returns inert text and never opens a URL automatically. Location is one-shot, contacts come only from Apple's single-contact picker, text-file import is bounded to one selected file, and pedometer access returns today's aggregate steps. Voice notes stay local and are never uploaded or automatically transcribed. A linked transcript requires on-device recognition, never falls back to the network, and remains editable until you accept it. Sharing uses Apple's destination chooser; clipboard access is tap-initiated and write-only. There is no background camera, microphone, speech, location, contact, file, or motion access.

How do I open a tiny app from Shortcuts or Siri?

Generate or revise that tiny app with one Shortcuts access block and approve the capability. In Apple's Shortcuts app, add MakeYour's Open Tiny App action and choose the project. Its picker receives only opted-in projects' stable IDs, names, and safe icons. Local device authentication is required, and the action only opens MakeYour in the foreground.

If a saved shortcut says the tiny app is unavailable, the project was deleted, its access block was removed, or the local catalog failed safe. Duplicates do not inherit the opt-in. Add the block again only when you want to re-enable access.

Does AI see photos or records?

No. A tiny-app AI helper sends only text entered or visibly prefilled into that helper after the exact task and text are shown for confirmation. An accepted transcript may become an editable prefill, but audio is never attached. Photos, records, scans, coordinates, contacts, imported files, step counts, other apps, and general device data are not attached automatically.

Delete my data

Long-press an app in My Apps and choose Delete to remove its document, project-local state and images, local notifications, and Shortcuts eligibility. Remove host-level OpenAI and Twelve Data keys with the in-app controls before uninstalling. Uninstall removes sandboxed projects, assets, caches, and runtime state, but does not guarantee deletion of Keychain items.
