MakeYour Support

Help for building and using your private native tiny apps.

Contact

Email support@claude-world.com. Include the MakeYour version, iOS version, feature, and exact error text. Never send an API key or private content.

Where is my API key?

Your OpenAI key is stored in iOS Keychain with When Unlocked, This Device Only protection. Remove it from AI Key → Remove. An optional Twelve Data key for symbols beyond its AAPL demo uses the same protection and can be removed from the market provider-key screen.

Generation is not working

Confirm your key is valid for the selected Responses API model, Allow requests to OpenAI is enabled, and the iPhone is online. Provider errors appear in the app without printing the key.

Are FX values live market quotes?

No. They are the latest available daily reference rates from Frankfurter. They are not streaming quotes or financial advice.

Where do news and stock data come from?

News uses fixed BBC World, BBC Technology, and NPR News RSS feeds; local search, topics, and bookmarks are not sent to those providers. Market data uses fixed Twelve Data endpoints. AAPL supports demo access; other symbols require your own Twelve Data key. Quotes can be delayed or rate-limited and are not financial advice.

Why is a device action unavailable?

Camera capture, live QR/barcode/text scanning, pedometer data, and meaningful haptics require supported physical iPhone hardware and may be unavailable in Simulator. Camera, location, and motion also require the corresponding iOS permission. MakeYour starts these actions only after you tap.

What can device actions access?

Scanning returns inert text and never opens a URL automatically. Location is one-shot, contacts come only from Apple's single-contact picker, text-file import is bounded to one selected file, and pedometer access returns today's aggregate steps. Sharing uses Apple's destination chooser; clipboard access is tap-initiated and write-only. There is no background camera, location, contact, file, or motion access.

Does AI see photos or records?

No. A tiny-app AI helper sends only text entered into that helper after the exact task and text are shown for confirmation. Photos, records, scans, coordinates, contacts, imported files, step counts, other apps, and general device data are not attached automatically.

Delete my data

Long-press an app in My Apps and choose Delete to remove its document, project-local state and images, and local notifications. Remove host-level OpenAI and Twelve Data keys with the in-app controls before uninstalling. Uninstall removes sandboxed projects, assets, caches, and runtime state, but does not guarantee deletion of Keychain items.
