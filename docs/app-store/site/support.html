MakeYour Support

Help for building and using your private native mini apps.

Contact

Email support@claude-world.com. Include the MakeYour version, iOS version, feature, and exact error text. Never send an API key or private content.

Where is my API key?

It is stored in iOS Keychain with When Unlocked, This Device Only protection. Remove it from AI Key → Remove.

Generation is not working

Confirm your key is valid for the selected Responses API model, Allow requests to OpenAI is enabled, and the iPhone is online. Provider errors appear in the app without printing the key.

Are FX values live market quotes?

No. They are the latest available daily reference rates from Frankfurter. They are not streaming quotes or financial advice.

Does AI see photos or records?

No. A mini-app AI helper sends only text entered into that helper after the exact task and text are shown for confirmation. Photos, records, other apps, and device data are not attached.

Delete my data

Long-press an app in My Apps and choose Delete. Remove the API key from AI Key. Deleting MakeYour removes all remaining local data.
