## Description

```text
MakeYour is one native iPhone app where you can create, use, and evolve the personal tiny apps you need.

Add your own OpenAI API key, describe a tool, and GPT-5.6 returns a strict, versioned AppDocument. MakeYour validates it and renders it through precompiled SwiftUI components and allowlisted native capabilities. It never downloads or executes generated Swift, JavaScript, plug-ins, or arbitrary code.

This real iOS Simulator demo shows:
• My Apps, Builder, and AI Key working as one personal app workshop
• A complex TripPilot travel command center generated from one prompt
• A first candidate failing semantic validation, followed by automatic validator-guided repair revision 1
• Capability review and the completed native tiny app opening successfully
• Its working generated converter: 100 USD = 3,250 TWD
• Another tiny app being restyled and extended while its completed task remains intact
• Switching among multiple tiny apps with different interfaces, capabilities, and local data
• How Codex built and tested the schema, validator, repair loop, safety boundary, and SwiftUI runtime

The TripPilot generation, repair, capability review, app launch, and currency result are real. Only the model waiting intervals are accelerated and clearly labeled. No API key is exposed. TripPilot is generated from the prompt; it is not a hard-coded template.
```
