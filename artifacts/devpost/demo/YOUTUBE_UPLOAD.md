## Description

```text
MakeYour is one native iPhone app for creating, using, and evolving the personal mini apps you need.

Add your own OpenAI API key, describe a tool, and GPT-5.6 returns a strict, versioned AppDocument. MakeYour validates that document and renders it through precompiled SwiftUI components and allowlisted native capabilities. It never downloads or executes generated Swift, JavaScript, plug-ins, or arbitrary code.

This real iOS Simulator demo shows:
• My Apps, Builder, and AI Key working as one personal app workshop
• Daily Focus generated from a real prompt
• The same app restyled and extended from Version 2 to Version 3 while a completed task remains intact
• Switching among a travel budget converter, live currency alerts, a private food journal, and other mini apps
• How Codex helped build and test the schema, validator, safety boundary, and native SwiftUI runtime

Each mini app can have its own interface, allowlisted capabilities, and local data while remaining inside MakeYour. No separate Xcode project or per-app build pipeline is created.

The generation wait is shortened and clearly labeled. The prompt, validation result, generated interface, retained task state, and app switching are real Simulator workflows. No API key is exposed.
```
